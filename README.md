##### 编译时报错找不到符号  
  在IDEA Plugin界面搜索安装 `Lombok` 插件  
  点击 `Setting` -> `Build,Execution,Deployment` -> `Compiler` -> `Annotation Processors` 
  选择项目  
  勾选 `Enable annotation processing`、`Obtain processors from project classpath`
